# ProyectoSoftwareFJ
Sistema desarrollado en Python aplicando Programación Orientada a Objetos.

El proyecto implementa:
- Manejo de excepciones
- Validaciones
- Herencia
- Abstracción
- Encapsulación
- Registro de errores mediante logs
- Control de estabilidad del sistema
- Uso de Git y GitHub

Desarrollado para la Fase 4 del curso de programación
